error in can accept error information wired from VIs previously called. Use this information to decide if any functionality should be bypassed in the event of errors from other VIs.

Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

    		status

    status is TRUE (X) if an error occurred or FALSE (checkmark) to indicate a warning or that no error occurred.

    Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

    		code

    code is the error or warning code.

    Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

    source describes the origin of the error or warning.

    		AO 6711 Inputs

      		Physical Channels

      Specify the names of the physical channels to use to create virtual channels. The DAQmx physical channel constant lists all physical channels on devices and modules installed in the system.

      To specify a range of physical channels, use a colon between two channel numbers or two physical channel names:

      Dev1/ao0:3

      Dev1/ao0:Dev1/ao3

      You can specify channel ranges in reverse order:

      Dev1/ao4:0

      Use commas to separate physical channel names and ranges in a list as follows:

      Dev1/ao0, Dev1/ao2:3

      		Min Voltage

      Specify the minimum value you will generate.

      This value is used to help determine the range of the DAC. The actual range will be coerced to one that your hardware supports. Refer to your device documentation for a list of available ranges.

      		Max Voltage

      Specify the maximum value you will generate.

      This value is used to help determine the range of the DAC. The actual range will be coerced to one that your hardware supports. Refer to your device documentation for a list of available ranges.

      		counter

    		AWG 5421 Inputs

      		Resource Name

      Pass the resource name of the NI FGEN device to initialize.

      Valid Range:

      DAQ::x - x is the NI SCOPE device number obtained from the NI-DAQ Configuration Utility.

      		Sample Rate

      Pass the sample rate at which you want the function generator to output arbitrary waveforms. The driver sets the niFgen>>Arbitrary Waveform Output>>Sample Rate property to this value.

      Units: Samples-per-second (Samp/s)

      Default Value: None

      		Trigger Type

      Specify the trigger source you want to use. The driver sets the

      niFgen>>Triggering Attributes>>Trigger Source property to this

      value.

      After you call the niFgen Initiate Generation VI, the function generator waits for the trigger you specify in this parameter. After it receives a trigger, the function generator produces the number of cycles you specify in the Cycle Count property.

      This is also the source for the trigger in the other trigger modes as specified by niFgen Configure Trigger Mode

      Notes:

      (1) Immediate - The function generator does not wait for a trigger of any kind.

      (2) External - The function generator waits for a trigger on the external trigger input.

      (3) SW Trig Func - The function generator waits until you call the niFgen Send Software Trigger VI.

      		Trigger Source

      Source specifies which trigger source the signal generator uses.

    		DIGIT 5122 Inputs

      		Sample Rate

      Specify the minimum sampling rate for the acquisition.

      Units: Hertz (Samples / Second)

      Valid Range:

      The combination of sampling rate and minimum record length must allow the scope to sample at a valid sampling rate for the acquisition type specified in niScope Configure Acquisition and not require more memory than the onboard memory module allows.

      		Resource Name

      Pass the resource name of the NI SCOPE device to initialize.

      Valid Range:

      DAQ::x - x is the NI SCOPE device number obtained from the NI-DAQ Configuration Utility.

      		Channel Name

      Pass the channel name you want to configure.

      Valid Values:

      0..n-1, n is the number of channels on the NI SCOPE device

      		Trigger Source

      Pass the source you want the digitizer to monitor for a trigger.

      Default Value: Channel 0

      		Vertical Range

      Pass the value of the input range the oscilloscope uses for the channel. For example, to acquire a sine wave that spans between -5.0 and 5.0 volts, pass 10.0 as the value of this parameter.

      Units: Volts

      Valid Range:

      Check the Hardware User's Manual for a device's list of supported vertical ranges. If the specified range is not supported by a device, the value will be coerced up to the next valid range.

      		Trigger Level

      Specify the voltage threshold for the trigger.
